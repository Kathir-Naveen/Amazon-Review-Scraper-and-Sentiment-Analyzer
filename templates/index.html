About Amazon Review Analyzer

This application allows you to analyze sentiment in Amazon product reviews.

Simply enter an Amazon product URL, and the application will:

  • Scrape all available reviews from the product page
  • Extract product details and image
  • Analyze the sentiment of reviews using VADER
  • Display results using an emoji-based sentiment meter
  • Generate a word cloud of most frequent terms

Created as a final year project in 2025.
